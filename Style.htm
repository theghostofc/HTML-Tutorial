STYLE

This tag is included in the HEAD Tag. It contains the Style Sheet Definitions . Place the contents between the !--...-- Tag to keep the Style Hidden from the Browsers which don't support Scripts or are forced to do that. This is a Container Tag and should be closed.

Attributes

  1. MEDIA : It specifies the Destination medium for the Style Information. It may be a Single Type or a Comma separated List .
    1. SCREEN : It is for Online Viewing of the Document. It is the Default .
    2. PRINT : It is for Traditional Printed Material and for Documents to be Viewed in the Print Preview Mode .
    3. PROJECTION : It is for viewing on Projectors .
    4. BRAILLE : It is for Braille Tactile Feedback Devices .
    5. SPEECH : It is for a Speech Synthesizer .
    6. ALL : It Applies to All Devices .
  2. TYPE : It specifies the MIME Type of the Style Sheet Specification Standard used.

Syntax

<STYLE [MEDIA="MEDIA"] [TYPE="TYPE"]>
<!--
<tagname> {
<property1>:<specifications>;
<property2>:<specifications>;
}
-->
</STYLE>

Example

<STYLE MEDIA="ALL" TYPE="TEXT/CSS">
<!--
H1 {
BACKGROUND:TEAL;
COLOR:CYAN;
}
-->
</STYLE>
Effect
The Effect can't be shown on the Document now. This entire Document is Composed by Style.
