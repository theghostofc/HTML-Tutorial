TH

This tag is included in the TR Tag. It contains the Table Cell Headings i.e., Table Head .
This is a Container Tag and can be closed. The Closing is Optional.

Attributes

  1. ABBR : It specifies Short Replacement Text associated with the tag contents. .
  2. ALIGN : It specifies the Alignment of the text of the Heading.
    1. LEFT : It will Left Align the text of the Heading.
    2. RIGHT : It will Right Align the text of the Heading.
    3. CENTER : It will Center Align the text of the Heading.
    4. JUSTIFY : It will Justify Align the text of the Heading.
    5. CHAR : It will specify that the alignment to be done on the basis of a Character .
  3. CHAR : It specifies the Character on the basis of which the alignment of the text will be done. The Default is the Dot '.' .
  4. CHAROFF : It specifies the number of Characters from the left at which the alignment appears .
  5. COLSPAN : It specifies that a Table Cell occupies more than One Column . It is mostly used if some Category Name applies to more than One Column Data.
  6. NOWRAP : It specifies that the text within the Table Cell in the row does not Wrap. This may cause the table to expand beyond the horizontal dimensions of the current page.
  7. ROWSPAN : It specifies that a Table Cell occupies more than One Row . It is useful if several rows of information relate to One Category.
  8. VALIGN : It specifies the Vertical Alignment of the text of the Row.
    1. TOP : It will align the text with the Top of the Row.
    2. MIDDLE : It will align the text with the Middle of the Row.
    3. BOTTOM : It will align the text with the Bottom of the Row.
    4. BASELINE : It will align the text with the Baseline of the Row.
    5. WIDTH : It specifies the Horizontal Dimesions of the cellin number of Pixels or as a Percentage of the Table Width .

Syntax

<TABLE>
<TR>
<TH>
...Content...
</TH>
</TR>
</TABLE>

Example

<TABLE>
<TR>
<TH>
This is a Table Heading
</TH>
</TR>
<TR>
<TD>
This is a Table Data
</TD>
</TR>
</TABLE>
Effect
This is a Table Heading
This is a Table Data
