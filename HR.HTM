HR

This is to be used in the BODY Tag. The Browser draws a Horizontal Row depending upon the width specified. This is not a Container Tag and thus need not be closed.

Attributes

  1. SIZE : Specifies the horizontal size of the row.
  2. WIDTH : Specifies the vertical size of the row.
  3. NOSHADE : Specifies whether the row is to be filled or not .

Syntax

X in %.
<HR [SIZE=X] [WIDTH="X%"] [NOSHADE]>
X in Pixels.
<HR [SIZE=X] [WIDTH=X] [NOSHADE]>

Example

X in %.
<HR SIZE= 50 WIDTH="50%" NOSHADE>
X in Pixels.
<HR SIZE= 50 WIDTH= 50 NOSHADE>
