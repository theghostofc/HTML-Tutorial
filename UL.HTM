UL

This is to be used in the BODY Tag. This tag is used to create Unordered or Bulleted Lists .
This is a Container Tag and thus has to be closed.

Attributes

  1. TYPE :
    1. "disc" : To create lists as


      ....       (This is the Default Type )
    2. "circle" : To create lists as


      ....
    3. "square" : To create lists as


      ....

Syntax

<UL>
<LI> ...Text...
<LI> ...Text...
....
</UL>

Example

<UL>
<LI> List Item1
<LI> List Item2
</UL>
Effect

  • List Item1
  • List Item2
