COLGROUP

This specifies Attributes for a Group of Table Columns . This is to be used in the TABLE Tag. This is a Container Tag and thus needs to be closed.

Attributes

  1. ALIGN : It specifies the Alignment of the text of the Column Group.
    1. LEFT : It will Left Align the text of the Column Group.
    2. RIGHT : It will Right Align the text of the Column Group.
    3. CENTER : It will Center Align the text of the Column Group.
    4. JUSTIFY : It will Justify Align the text of the Column Group.
    5. CHAR : It will specify that the alignment to be done on the basis of a Character .
  2. CHAR : It specifies the Character on the basis of which the alignment of the text will be done. The Default is the Dot '.' .
  3. CHAROFF : It specifies the number of Characters from the left at which the alignment appears .
  4. VALIGN : It specifies the Vertical Alignment of the text of the Column Group.
    1. TOP : It will align the text with the Top of the Row.
    2. MIDDLE : It will align the text with the Middle of the Row.
    3. BOTTOM : It will align the text with the Bottom of the Row.
    4. BASELINE : It will align the text with the Baseline of the Row.
  5. WIDTH : Specifies the Width or Horizontal Dimension of the Column Group ( In px or % ) .

Syntax

<TABLE>
<COLGROUP>
<COL>
...
<TR>
<TD> Contents </TD>
...
</TR>
...
</COLGROUP>
...
</TABLE>

Example

<TABLE BORDER=1>
<COLGROUP>
<COL ALIGN="LEFT">
<COL ALIGN="CENTER">
<COL ALIGN="RIGHT">
<TR>
<TD> This Cell is Left Aligned </TD>
<TD> This Cell is Center Aligned </TD>
<TD> This Cell is Right Aligned </TD>
</TR>
</COLGROUP>
</TABLE>
Effect
This Cell is Left Aligned This Cell is Center Aligned This Cell is Right Aligned
