SUP

This is to be used in the BODY Tag. The text written in this tag is displayed as a Superscript
by the Browser. This is a Container Tag and thus has to be closed.

Syntax

<SUP> ...Text... </SUP>

Example

<SUP> This is a Text. This is displayed as a Superscript. </SUP>
Effect
See this Text. This is displayed as a Superscript.
