TEXTAREA

This is to be used in the FORM Tag. This Tag defines a Multiple Line Text Input Field in a Form. To specify a Default Value , place the text within the opening and the closing tags. This is a Container Tag and thus need to be closed.

Attributes

  1. COLS : Indicates the Width (in Character Widths) of the Text Input Field.
  2. DATAFLD : Selects a Column from a Previously Identified Source of Tabular Data .
  3. DATASRC : Specifies the Source of Data for Data Binding .
  4. DISABLED : Denies Access to the Input Field.
  5. NAME : Gives a Name to the value you pass to the Form Processor.
  6. READONLY : Specifies that the Visitor can't Change the contents of the Text Input Field.
  7. ROWS : Indicates the Height (in Lines of Text) of the Text Input Field.

    Syntax

    <FORM [NAME="NAME"] >
    <TEXTAREA [NAME="TNAME"] [COLS="n"] [ROWS="m"]>
    [...Default Value...]
    </TEXTAREA>
    </FORM>

    Example

    <FORM [NAME="NAME"] >
    <TEXTAREA NAME="TXTAREA" COLS="20" ROWS="4">
    Comments
    </TEXTAREA>
    </FORM>
