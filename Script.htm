SCRIPT

This is to be used in the HEAD Tag. This contains the Browser Script Code . The popular ones
are JAVASCRIPT and VBSCRIPT . It is good habit to include the actual Script Code in the
!--...-- Tags so that the browsers that don't Scripts can ignore it. This is a Container Tag and has to be Closed.

Attributes

  1. DEFER : Indicates browser that the Script doesn't affect the initial document display, so that the Script can be processed after the page loads.
  2. LANGUAGE : Indicates the MIME Type of the Script.
  3. SRC : Specifies the Relative or Absolute URL of the Script to be included in the document.
  4. TYPE : Indicates the MIME Type of the Script. This is used as an alternate for the LANGUAGE Attribute.

Syntax

<HEAD>
<SCRIPT [DEFER][LANGUAGE="LANGUAGE"][SRC="URL"][TYPE="TYPE"]>
<!--
....Script....
//-->
</SCRIPT>
</HEAD>

Example

<HEAD>
<SCRIPT LANGUAGE="JAVASCRIPT" SRC="JAVA.JS">
<!--
function() {
....
}
//-->
</SCRIPT>
</HEAD>
Effect
The Effect can't be shown on the Document. But the Script will be included in the document.
Refer to the !--...-- Tag.
