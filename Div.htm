DIV

This is to be used in the BODY Tag. Indicates a Logical Division within a Document. Use the SPAN Tag to apply styles at a Small Level. This is a Container Tag and thus needs be CLosed.

Syntax

<DIV>
...Text/Image...
</DIV>

Example

This is a Text in some format.
<DIV>
This is a Text in other format defined in div.
</DIV>
Effect
This is a Text in some format.
This is a Text in other format defined in div.
