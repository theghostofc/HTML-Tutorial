MARQUEE

This is to be used in the BODY Tag. The text written in this tag Scrolls as specified. This only works on Internet Explorer & NOT on Netscape Navigator. This is a Container Tag and thus has to be closed.

Attributes

  1. BEHAVIOR
    1. ALTERNATE : Scroll Horizontally and return back after end of screen is encountered.
    2. SLIDE : Slide Horizontally only Once & doon't return back.
    3. SCROLL : Scroll Horizontally and return back from the same side.
  2. DIRECTION
    1. UP : Setup the direction of scrolling as bottom to top.
    2. DOWN : Setup the direction of scrolling as top to bottom.
    3. LEFT : Setup the direction of scrolling as right to left.
    4. RIGHT : Setup the direction of scrolling as left to right.
NOTE : Only one out of Behavior & Direction can be used at a time.

Syntax

<MARQUEE [BEHAVIOR="ATTRIBUTES"] [DIRECTION="ATTRIBUTES"] ...Text / Image... </MARQUEE>

Example

<MARQUEE BEHAVIOR="ALTERNATE"> ...Text / Image... </MARQUEE>

<MARQUEE DIRECTION="RIGHT"> ...Text / Image... </MARQUEE>
