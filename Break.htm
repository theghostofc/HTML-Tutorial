BR

This is to be used in the BODY Tag. This breaks the current line and starts a new line. This is not a Container Tag and thus need not be closed.

Attributes

  1. CLEAR :
    1. LEFT : Break to the next line only when the left margin is clear.
    2. RIGHT : Break to the next line only when the right margin is clear.

Syntax

...Text...
<BR>
...Text...
OR
...Text / Image...
<BR [CLEAR="LEFT / RIGHT"]>
...Text / Image...

Example

<BR>
<BR CLEAR="LEFT">
Effect
Current Line Breaks
To Next Line
