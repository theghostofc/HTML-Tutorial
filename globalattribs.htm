GLOBAL ATTRIBUTES

These Attributes work with Almost all Tags .

Attributes

  1. ALIGN : Indicates the Alignment of the Value of the Tag.
    1. CENTER : Align Center.
    2. LEFT : Align Left.
    3. RIGHT : Align Right.
      NOTE : Use of ALIGN with many attributes has been depricated in HTML 4.0.
  2. ACCESSKEY : Assigns a Keyboard Key to the Tag or its Elements .It with the ALT Key i.e., ALT+KEY.
  3. CLASS : Indicates the Style Class to be Applied to the Tag .
  4. ID : Assigns a Unique ID Selector to an Instance of the Tag. When you then assign a Style to that ID selector, it affects only that instance of the Tag .
  5. STYLE : Specifies the CSS Commands/Properties that applies to the contents of the Tags .
  6. TABINDEX : Indicates where the Element appears in the Tabing order of the Document .
  7. TITLE : Browsers Use this text to Display a ToolTip Over the Contents of the Tag .

Syntax

  1. ALIGN="{CENTER|LEFT|RIGHT}"
  2. ACCESSKEY="n"
  3. CLASS="Name"
  4. ID="Name"
  5. STYLE="Style Sheet Commands"
  6. TABINDEX="n"
  7. TITLE="Text"
