DEL

This is to be used in the BODY Tag. It indicates Text Marked for Deletion from the Document . This is a Container Tag and thus needs be CLosed.

Attributes

  1. CITE : Indicates Address of Reference for Insertion .
  2. DATETIME : Indicates the Date and Time in Precisely this Format : YYYY-MM-DDThh:mm:ss-hu:thTZD Or YYYY-MM-DDThh:mm:ssZ . hu, th represent Hundredth and Thousandth of a Second Respectively .

Syntax

<DEL [CITE="URL"] [DATETIME="DATETIME"]>
....Text Marked for Deletion....
</DEL>

Example

The Following Text is to be
<DEL DATETIME="2001-12-29T17:33:43Z">
Deleted from
</DEL>
this Place.
Effect
The Following Text is to be Deleted from this Place.
