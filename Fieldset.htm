FIELDSET

This is to be used in the BODY Tag. It is used to group related content in a Document or
Web Page. . The LEGEND Tag may be used to give the group a Suitable Heading. This is a Container Tag and thus needs be CLosed.

Syntax

<FIELDSET>
<LEGEND>
....Other Tags....
</FIELDSET>

Example

<FIELDSET>
<LEGEND>
...Form Input(s) (Used Generally...
</FIELDSET>
