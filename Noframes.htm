NOFRAMES

This is to be used after the FRAMESET Tag. This Tag provides HTML content for browsers
that don't support frames or are configured to not to do so. The BODY Tag can be included
in this Tag to provide Additional Formatting or CSS Features in the Document. That too will be displayed on the browser if it supports it. This is a Container Tag and thus has to be closed.

Syntax

<NOFRAMES>
...Text / Any Tag...
</NOFRAMES>

Example

<NOFRAMES>
If you see this Message again in the EFFECT Part, Your Browser Don't support Frames.
</NOFRAMES>
Effect
If you see this Message again in the EFFECT Part, Your Browser Don't support Frames.
