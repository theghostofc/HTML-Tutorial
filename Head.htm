HEAD

This Tag gives the browser all the control information of a Web Page. All the Global Settings are defined here. The scripting functions are also defined here. This is also a Container Tag and thus has to be closed. Text Written in the HEAD Tag, except of what is written in the TITLE Tag
is not displayed on the Browser Window.

Syntax


<HTML>
<HEAD>
...
...
...
</HEAD>
...
...
</HTML>
