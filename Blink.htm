BLINK

This is to be used in the BODY Tag. The text written in this tag Blinks.
Its effect can be seen on Netscape Navigator & NOT on Internet Explorer. This is a Container Tag and thus has to be closed.

Syntax

<BLINK> ...Text... </BLINK>

Example

<BLINK> This will Blink on Netscape Navigator </BLINK>
Effect
This will Blink on Netscape Navigator
