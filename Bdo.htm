BDO

This is to be used in the BODY Tag. This Tag Indicates the Direction of the Text.
This is a Container Tag and thus needs to be closed.

Attributes

  1. LTR : To Make the Direction Left To Right.
  2. RTL : To Make the Direction Right To Left.

Syntax

<BDO="ATTRIBUTES">
...Text...
</BDO>

Example

<BDO="LTR">
Text Appearing as Left To Right
</BDO>
Effect
Text Appearing as Left To Right
