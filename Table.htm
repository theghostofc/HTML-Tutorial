TABLE

This tag is included in the BODY Tag. Specifies a Container Table within your doocument .
This is a Container Tag and should be closed.

Attributes

  1. ALIGN : It specifies the Alignment of the text of the Table.
    1. LEFT : Positions the Table on the Left of the Window .
    2. RIGHT : Positions the Table on the Right of the Window .
    3. CENTER : Positions the Table on the Center of the Window .
  2. CELLPADDING : It specifies the space (in the Number of Pixels) between the edges of the table cells and their contents .
  3. CELLSPACING : It specifies the space (in the Number of Pixels) between the borders of the table cells and the borders of their adjacent cells.
  4. COLS : It specifies the Number of Columns in the Table.
  5. FRAME : It specifies the Vertical Alignment of the text of the Column.
    1. VOID : Indicates No Borderlines .
    2. BOX : Indicates Borders around the Entire Table. This is the Default .
    3. ABOVE : Indicates Borderline Above the Top Edge .
    4. BELOW : Indicates Borderline Along the Bottom Edge .
    5. HSIDES : Indicates Borderline Along the Top and the Bottom Edges .
    6. LHS : Indicates Borderline Along the Left Side .
    7. RHS : Indicates Borderline Along the Right Side .
    8. VSIDES : Indicates Borderline Along the Left and the Right Sides .
  6. RULES : It specifies where the Rule Lines appear inside the Table. For the RULES attribute to work, set the BORDER Attribute .
    1. NONE : Indicates No Rule lines .
    2. ROWS : Indicates Rule lines between Rows .
    3. COLS : Indicates Rule lines between Columns .
    4. ALL : Draws All possible Rule Lines .
    5. GROUP : Specifies Rule lines between the Groups Defined by the TFOOT , THEAD , TBODY and COLGROUP Tags.
  7. SUMMARY : It specifies descriptive text for the Table. This is used to summarize the Table for display on the browsers That don't support Graphics or Tables.
  8. WIDTH : It specifies the Width of the Table. This is used to specify the size of the Table w.r.t. the browser's size.

Syntax

<TABLE>
....
</TABLE>

Example

<TABLE>
<TR>
<TD> This is the Table Data </TD>
</TR>
</TABLE>
Effect
This is the Table Data
