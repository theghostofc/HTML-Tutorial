LINK

This is to be used in the HEAD Tag. This Tag Eshtablishes Relationship between Current Document and other Documents. This is not a Container Tag and thus need not be closed.

Attributes

  1. HREF : Specifies the Relative or Absolute Address (URL) of the file to be linked.
  2. HREFLANG : Specifies the Language used in the document linked to. Use Standard codes for languages. Visit http://www.oasis_open.org/cover/iso639.html for more
    Information regarding Language Codes.
  3. MEDIA : It specifies the Destination medium for the Style Information. It may be a Single Type or a Comma separated List .
    1. SCREEN : It is for Online Viewing of the Document. It is the Default .
    2. PRINT : It is for Traditional Printed Material and for Documents to be Viewed in the Print Preview Mode .
    3. PROJECTION : It is for viewing on Projectors .
    4. BRAILLE : It is for Braille Tactile Feedback Devices .
    5. SPEECH : It is for a Speech Synthesizer .
    6. ALL : It Applies to All Devices .
  4. REL : It defines the Relation between the current document and another resource.
    1. TOP : It defines the Homepage of the Site or the Top of the Site Hierarchy
    2. CONTENTS : It defines the Location of a resource that lists the contents of the Site.
    3. INDEX : It Provides a link to the Index of the Site.
    4. GLOSSARY : It indicates the location of a Glossary Resource.
    5. COPYRIGHT : It indicates the Location of a Copyright Statement.
    6. NEXT & PREVIOUS : It eshtablishes Relationships between documents or resources in series.
    7. HELP : It indicates the Location of a Help resource.
    8. SEARCH : It specifies the Location of a Search resource.
    9. STYLESHEET : It specifies information about CSS.
  5. REV : It eshtablishes Reverse Relationships between the current document and other resources .
  6. TYPE : It specifies the MIME Type of the Style Sheet Specification Standard used.

Syntax

<HEAD>
<LINK [HREF="URL"] [HREFLANG="URL"] [MEDIA="MEDIA"] [REL="RELATION"] [REV="REV. REL."] [TYPE="MIME TYPE"]>
.....
</HEAD>

Example

<HEAD>
<LINK HREF=".\CSS\IMP.CSS" REL=STYLESHEET TYPE="TEXT/CSS">
.....
</HEAD>
Effect
The Effect can't be shown on the Document.
