CITE

This is to be used in the BODY Tag. The text written in this tag is displayed as a Bibliographic Citation by the Browser. This is a Container Tag and thus has to be closed.

Syntax

<CITE> ...Text... </CITE>

Example

<CITE> This is a Text. This is displayed as a bibliographic citation. </CITE>
Effect
This is a Text. This is displayed as a bibliographic citation.
