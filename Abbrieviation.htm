ABBR

This is to be used in the BODY Tag. The text written in this tag is displayed as an Abbreviation
by the Browser. This is a Container Tag and thus has to be closed.

Syntax

<ABBR> ...Text... </ABBR>

Example

<ABBR> H.T.M.L. </ABBR> Hyper Text Markup Language
Effect
H.T.M.L. Hyper Text Markup Language
