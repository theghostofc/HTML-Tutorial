DD

This is to be used in the DL Tag. The Data Definition is enclosed in this tag. This is a Container Tag and thus has to be closed.

Syntax

<DL>
<DT> ...Term... </DT>
<DD> ...Definition... </DD>
</DL>

Example

<DL>
<DT> Term1 </DT>
<DD> This the Definition for Term1. </DD>
</DL>
Effect

Term1

This the Definition for Term1.
