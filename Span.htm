SPAN

This tag is included in the BODY Tag. It defines Inline Sections for a Document affected by Style Sheet Attributes. . Use the DIV Tag to apply styles at the Block Element Level.
This is a Container Tag and should be closed.

Syntax

<SPAN>
...Text/Image...
</SPAN>

Example

This is a Text in some format.
<SPAN>
This is a Text in other format defined in span.
</SPAN>
Effect
This is a Text in some format.
This is a Text in other format defined in span.
