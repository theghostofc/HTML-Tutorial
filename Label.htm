LABEL

This is to be used in the BODY Tag. It specifies Identifies Text For a Form Widget . This is a Container Tag and thus needs be CLosed.

Attributes

  1. DISABLED : Denies Access to the Input Method.
  2. FOR : Specifies the ID of the Widget Associated with the Label.

Syntax

<LABEL [DISABLED] [FOR="FORM NAME"]>
....Label For The Form....
</LABEL>

Example

<LABEL [DISABLED] [FOR="FORM NAME"]>
This is the Label for the First Form
</LABEL>
