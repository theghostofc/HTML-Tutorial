PARAM

This is to be used in the OBJECT Tag. This specifies Parameters to be passed to an
Embedded Object . This is not a Container Tag and thus need not be closed.

Attributes

  1. NAME : Indicates the Name of the Parameter passed to the Embedded Object.
  2. TYPE : Specifies the MIME type of the Data found at the specified URL. Use with the [VALUETYPE="REF"].
  3. VALUE : Specifies the value associated with the Parameter passed to the Embedded Object.
  4. VALUETYPE :
    1. REF : Indicates a URL passed to the Embedded Object.
    2. OBJECT : Indicates that the Value Attribute specifies the location of the Object Data.
    3. DATA : Indicates that the Value Attribute is set to Plain Text String. It is used for passsing alphanumeric data to the Embedded Object.

Syntax

<OBJECT>
<PARAM [NAME="NAME"] [TYPE="TYPE"] [VALUE="VALUE"] [VALUETYPE="OPTIONS"]>
</OBJECT>

Example

<OBJECT>
<PARAM NAME="PARAM1" TYPE="IMAGE/GIF" VALUE="IMAGES\GRADUATION.GIF" VALUETYPE="OBJECT">
</OBJECT>
Effect
The Effect can't be shown on the Document Now.
