TITLE

This tag is included in the HEAD portion of the page. It is shown on the TITLE bar of the browser window. Default is the filename of the HTML document. There can be only on TITLE for a page. The TITLE can't use the formatting images and links to other pages. Don't use ':' and '\' in the TITLE of the page. This is a Container Tag and thus to be closed.

Syntax

<TITLE> ...TITLE... </TITLE>

Example

<TITLE> My First Web Page </TITLE>
Effect
The Effect can't be shown on the Document. Its Effect is Visible on the TITLE Bar of this Browser Window.
