THEAD

This tag is included in the TABLE Tag. It defines the Table Header . It should contain atleast one TR Tag. This is a Container Tag and should be closed.

Attributes

  1. ALIGN : It specifies the Alignment of the text of the Table Header.
    1. LEFT : It will Left Align the text of the Table Header.
    2. RIGHT : It will Right Align the text of the Table Header.
    3. CENTER : It will Center Align the text of the Table Header.
    4. JUSTIFY : It will Justify Align the text of the Table Header.
    5. CHAR : It will specify that the alignment to be done on the basis of a Character .
  2. CHAR : It specifies the Character on the basis of which the alignment of the text will be done. The Default is the Dot '.' .
  3. CHAROFF : It specifies the number of Characters from the left at which the alignment appears .
  4. VALIGN : It specifies the Vertical Alignment of the text of the Table Header.
    1. TOP : It will align the text with the Top of the Row.
    2. MIDDLE : It will align the text with the Middle of the Row.
    3. BOTTOM : It will align the text with the Bottom of the Row.
    4. BASELINE : It will align the text with the Baseline of the Row.

Syntax

<TABLE>
<THEAD>
<TR>
<TD>
...Content...
</TD>
</TR>
</THEAD>
</TABLE>

Example

<TABLE>
<THEAD>
<TR>
<TD>
This is the Table Head
</TD>
</TR>
</THEAD>
<TR>
<TD>
This is a Row
</TD>
</TR>
</TABLE>
Effect
This is the Table Head
This is a Row
