FRAME

This is to be used in the FRAMESET Tag. It specifies what is to be displayed in the frame.
This is not a Container Tag and thus need not be closed.

Attributes

  1. NAME : Gives a Name to the Frame. It helps in the Identification of the frame.
  2. SRC : This specifies the URL and Filename of the file to be displayed on the browser window.
  3. NORESIZE : Specifies that the Frame can't be Resized. If it is not specified the user can resize the Frame as per his own requirement.
  4. SCROLLING : Specifies that the contents of the frame can be scrolled or not. It can have 3 Values.
    1. AUTO : Tells the browser to show the scroll bar only when the content is too big to be displayed at once in the frame.
    2. YES : This tells the browser to show the scroll bar even if the content fits in the frame. In that case the scroll bar is displayed as a Grayed Option .
    3. NO : This tells the browser to not to show the scroll bar even if the content does not fit in the window. In that case the scroll bar is not displayed and the content can be seen upto the size of the frame.
  5. TARGET : Specifies where the contents of the frame are to be displayed
    1. _BLANK : Tells the browser to show the contents of the frame in a New Window .
    2. _PARENT : This tells the browser to show the contents of the frame in the Parent Frameset or the Most Immiediate Frameset .
    3. _SELF : This tells the browser to show the contents of the frame in the Same Window This is the Default Type .
    4. _TOP : This tells the browser to break out of all frames & show in the Same Window .
    5. FRAMENAME : This allows the creator to display the contents in his own created frames .

Syntax

<FRAMESET [ATTRIBUTES]>
<FRAME [ATTRIBUTES]>
</FRAMESET>

Example

<FRAMESET ROWS="40%,60%">
<FRAMESET COLS="40%,60%">
<FRAME NAME="ONE" SCROLLING="AUTO" SRC="frameup.htm">
<FRAME NAME="TWO" SCROLLING="AUTO">
</FRAMESET>
<FRAMESET COLS="50%,50%">
<FRAME NAME="THREE" SCROLLING="YES" NORESIZE SRC="framedown.htm">
<FRAME NAME="FOUR" SCROLLING="NO" NORESIZE>
</FRAMESET>
Effect
