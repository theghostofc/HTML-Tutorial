!DOCTYPE

This is the First HTML Tag. This tells the Browser about the Document Type & how the document is to be handled & what version of HTML is used. This helps the browser to decide how the tags are to be handled & the information of the page is to be displayed. This is not a Container Tag and thus need not be closed.

Attributes

  1. HTML 4.0 TRANSITIONAL : Used for pages that still rely on deprecated tags such as FONT & BGCOLOR.
  2. HTML 4.0 : Used for pages that don't us the deprecated tags.
  3. HTML 4.0 FRAMESET : Used for pages using Frames.
  4. HTML X.0 : Used for pages using earlier versions of HTML. Specify the appropriate version no. instead of 'X'.

Syntax

<!DOCTYPE HTML idstring>

Example

<!DOCTYPE HTML Public "_//w3c//dtd HTML 4.0 //EN">
