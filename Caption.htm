CAPTION

This is to be used in the TABLE Tag. The text written in this tag is displayed as a Description for
a Table This is a Container Tag and thus has to be closed.

Attributes

  1. ALIGN : Indicates where the Caption appears.
    1. TOP : Top of the Table.
    2. BOTTOM : Bottom of the Table.
    3. LEFT : Left of the Table.
    4. RIGHT : Right of the Table.

Syntax

<TABLE>
<CAPTION> ...Text... </CAPTION>
</TABLE>

Example

<TABLE>
<CAPTION> Table Heading </CAPTION>
</TABLE>
