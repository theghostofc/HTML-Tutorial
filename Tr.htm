TR

This tag is included in the TABLE Tag. It contains a row of cells ie., Table Row in a table.
It contains the TH and TD Tags.
This is a Container Tag and can be closed. The Closing is Optional.

Attributes

  1. ALIGN : It specifies the Alignment of the text of the Row.
    1. LEFT : It will Left Align the text of the Row.
    2. RIGHT : It will Right Align the text of the Row.
    3. CENTER : It will Center Align the text of the Row.
    4. JUSTIFY : It will Justify Align the text of the Row.
    5. CHAR : It will specify that the alignment to be done on the basis of a Character .
  2. CHAR : It specifies the Character on the basis of which the alignment of the text will be done. The Default is the Dot '.' .
  3. CHAROFF : It specifies the number of Characters from the left at which the alignment appears .
  4. NOWRAP : It specifies that the text within the Table Cell in the row does not Wrap. This may cause the table to expand beyond the horizontal dimensions of the current page.
  5. VALIGN : It specifies the Vertical Alignment of the text of the Row.
    1. TOP : It will align the text with the Top of the Row.
    2. MIDDLE : It will align the text with the Middle of the Row.
    3. BOTTOM : It will align the text with the Bottom of the Row.
    4. BASELINE : It will align the text with the Baseline of the Row.

Syntax

<TABLE>
<TR>
<TD>
...Content...
</TD>
</TR>
</TABLE>

Example

<TABLE>
<TR>
<TD>
This is a Table
</TD> <TD>
With Two Columns
</TD>
</TR>
</TABLE>
Effect
This is a Table With Two Columns
