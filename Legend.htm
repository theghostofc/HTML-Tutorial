LEGEND

This is to be used in the FIELDSET Tag. It specifies Description for a Fieldset . This is a Container Tag and thus needs be CLosed.

Attributes

  1. ALIGN : Specifies the Alignment of the LEGEND w.r.t. the FIELDSET.
    1. TOP : Top of the Fieldset.
    2. BOTTOM : Bottom of the Fieldset.
    3. LEFT : Left of the Fieldset.
    4. RIGHT : Right of the Fieldset.

Syntax

<FIELDSET>
<LEGEND [ALIGN="ALIGNMENT"]>
....Other Tags....
</FIELDSET>

Example

<FIELDSET>
<LEGEND>
...Form Input(s) (Used Generally...
</FIELDSET>
