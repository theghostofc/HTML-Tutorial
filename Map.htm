MAP

This is to be used in the BODY Tag. This Tag specifies a Container for Client Side Image Maps. Use the AREA Tag in this Tag to specify the Dimensions of the Image Map. This is a Container Tag and thus has to be closed.

Attributes

  1. NAME : Eshtablishes a Name for the Map Information which can be later referenced by the USEMAP Attribute of the IMG Tag.

Syntax

<MAP [NAME="NAME"]>
<AREA>
.....
</MAP>
To use this tag please refer to the IMG Tag.

Example

<MAP NAME="MP1">
<AREA SHAPE="RECT" COORDS="1,1,100,100" HREF="HTTP://WWW.YAHOO.COM">
</MAP>
