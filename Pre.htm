PRE

This is to be used in the BODY Tag. The text written in this tag is displayed as it is by the Browser.
It is to write Preformatted Text. This is a Container Tag and thus has to be closed.

Syntax

<PRE> ...Text... </PRE>

Example

<PRE> This is an example of Preformatted Text </PRE>
Effect 
This is an		example of
		Preformatted
	Text
