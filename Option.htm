OPTION

Indicates Items in a fill out Form Selection List made by the SELECT Tag.
This is a Container Tag and thus has to be closed.

Attributes

  1. DISABLED : Denies Access to the Input Method.
  2. LABEL : Specifies Shorter Alternatives text assigned to this tag. Browsers use this to show Tool Tips over the Group.
  3. SELECTED : Marks a Selected List Item Preselected .
  4. VALUE : Indicates which Data is to be sent to the Form Processor if someone chooses the Item. If VALUE is not Present, the Text between the Starting and Ending Tags is Sent to the Form Processor .

Syntax

<FORM>
<SELECT>
<OPTION [LABEL="LABEL"][DISABLED][SELECTED][VALUE="VALUE"]>
Text to be shown on the Browser
</OPTION>
...
</SELECT>
</FORM>

Example

<FORM>
<SELECT>
<OPTION SELECTED VALUE="VALUE1">
Text1
</OPTION>
<OPTION VALUE="VALUE2">
Text2
</OPTION>
</SELECT>
</FORM>
