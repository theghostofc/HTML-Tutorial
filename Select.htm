SELECT

This is to be used in the FORM Tag. This specifies a Relation List within a Form.
Use the OPTION Tag to specify items in the Selection List. This is a Container Tag and thus has to be closed.

Attributes

  1. DISABLED : Denies Access to the Selection List .
  2. MULTIPLE : Indicate that a Visitor can select more than one Selection List Item at the same time.
  3. NAME : Gives a Name to the value passed to the Form Processor. This eshtablishes a Name-Value Pair with which the Form Processor can Work.
  4. SIZE : Specifies the Number of the Variable Items in the Selection List. If there are more items than those which are visible, a Scroll Bar provides access to the other items.

Syntax

<FORM>
<SELECT [NAME="NAME"][DISABLED][MULTIPLE][SIZE="n"]> ...
<OPTION>... ...</OPTION>
...
</SELECT>
</FORM>

Example

<FORM>
<SELECT NAME="SELE1">
<OPTION VALUE="VALUE1"> VALUE1 </OPTION>
<OPTIONVALUE="VALUE2"> VALUE2 </OPTION>
</SELECT>
</FORM>
