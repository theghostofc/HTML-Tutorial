OL

This is to be used in the BODY Tag. This tag is used to create Ordered Lists . This is a Container Tag and thus has to be closed.

Attributes

  1. TYPE :
    1. A : To create lists as
      A
      B
      C ....
    2. a : To create lists as
      a
      b
      c ....
    3. I : To create lists as
      I
      II
      III ....
    4. i : To create lists as
      i
      ii
      iii ....
    5. 1 : To create lists as
      1
      2
      3 ....
      (This is the Default Type )
  2. START :
    1. N : This specifies from which number the list is to be started with. To create lists of any type, the value of 'N' depends on the Type . For eg. for the type A B C ..., the value of N may comprise of any character from A-Z. If any no is specified, the No. will be automatically converted to that type. So one can specify the digits and the digits will be automatically converted to that type.

Syntax

<OL>
<LI> ...Text...
<LI> ...Text...
....
</OL>

Example

<OL>
<LI> List Item1
<LI> List Item2
</OL>
Effect

  1. List Item1
  2. List Item2
If the value of TYPE is 2 or more the LIST will start from that number.
