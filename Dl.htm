DL

This is to be used in the BODY Tag. This tag is used to create Definition Lists . Actually the list is embedded in this tag. This is a Container Tag and thus has to be closed.

Syntax

<DL>
<DT> ...
<DD> ....
<DT> ...
<DD> ....
</DL>

Example

<DL>
<DT> Term1
<DD> Definition1
<DT> Term2
<DD> Definition2
</DL>
Effect
Term1
Definition1
Term2
Definition2
