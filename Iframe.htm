IFRAME

This is to be used in the BODY Tag. This Tag creates Floating or Inline Frames within a Document. This is a Container Tag and thus has to be closed.

Attributes

  1. FRAMEBORDER : Indicates whether the Floating Frame has Visible Border. 0 means NoBorder and 1 means Border.
  2. HEIGHT : Specifies the Vertical Dimensions (in px.) of the Floating Frame.
  3. HSPACE : Specifies the Size of Margins (in px.) to the Left and Right of the Floating Frame.
  4. MARGINHEIGHT : Specifies the size of the Top and Bottom Margins (in px.) of the Floating Frame.
  5. MARGINWIDTH : Specifies the size of the Left and Right Margins (in px.) of the Floating Frame.
  6. NAME : Assigns the Frame A Unique Name. This is used to open a New Document in the same Frame.
  7. SCROLLING : Indicates whether the Floating Frame has Scroll Bars. NO means NoScrolling and Yes means Scrolling.
  8. SRC : Specifies the Relative or Absolute URL of the file to be loaded in the Floating Frame.
  9. VSPACE : Specifies the Size of Margins (in px.) to the Top and Bottom of the Floating Frame.
  10. WIDTH : Specifies the Horizontal Dimensions (in px.) of the Floating Frame.

Syntax

<IFRAME [FRAMEBORDER="{0,1}"] [HEIGHT="n"] [HSPACE="n"] [MARGINHEIGHT="n"] [MARGINWIDTH="n"] [NAME="NAME"] [SCROLLING="{YES,NO}"] [SRC="URL"] [VSPACE="n"] [WIDTH="n"]> </IFRAME>

Example

<IFRAME FRAMEBORDER="1" NAME="IFRM1" SCROLLING="YES" SRC="IMAGES\ani.GIF">
</IFRAME>
