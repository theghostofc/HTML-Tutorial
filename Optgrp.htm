OPTGRP

Specifies a description for a group of options SELECT Tag. This is a Container Tag and thus
has to be closed.

Attributes

  1. LABEL : Specifies Shorter Alternatives text assigned to this tag. Browsers use this to show Tool Tips over the Group.

Syntax

<FORM>
<SELECT>
<OPTGROUP [LABEL="LABEL"]>
<OPTION>
...Text...
</OPTION>
...
</OPTGROUP>
</SELECT>
</FORM>

Example

<FORM>
<SELECT>
<OPTGROUP>
<OPTION>
Text1
</OPTION>
<OPTION>
Text2
</OPTION>
</OPTGROUP>
</SELECT>
</FORM>
