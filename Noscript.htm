NOSCRIPT

This is generally to be used inside a Script Definition . This Tag provides HTML content for browsers that don't support script or are configured to not to do so. This is a Container Tag and thus has to be closed.

Syntax

<NOSCRIPT>
...Text / Any Tag...
</NOSCRIPT>

Example

<NOSCRIPT>
If you see this Message again in the EFFECT Part, Your Browser Don't support Script.
</NOSCRIPT>
