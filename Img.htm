IMG

This is to be used in the BODY Tag. This Tag places an Inline Image in a Document.
This is not a Container Tag and thus need not be closed.

Attributes

  1. ALT : Provides a Textual Description of the Image. It is useful for the Visitors who have text only browsers. It can also be used as a ToolTip.
  2. BORDER : Indicates the Width (in px.) of a Border around the Image. The Default value is 0 i.e., NoBorder.
  3. HEIGHT : Specifies the Vertical Dimensions (in px.) of the Image. If it is not used the Image is Displayed in its Original Height It is used with the Width Attribute to Enlarge or Reduce the Image.
  4. HSPACE : Eshtablishes a Margin of White Spaces (in px.) to the Left and Right of the Image.
  5. ISMAP : Indicates that the Graphic Image Functions as a Clickable Image Map. This attribute instructs the Browser to send the Pixel Coordinates to the Server Side Image Map or any CGI Application when the user selects the Image with a Mouse Pointer. This is basically to implement the Server Side Image Map but, now a days Client Side Image Maps are more Popular. See the USEMAP Attribute.
  6. LONGDESC : Provides a long Textual Description of the Image. It is useful for the Visitors who have text only browsers or for other reasons can't view the Image.
  7. SRC : Specifies the Relative or Absolute URL of the Image file which you want to load in your document.
  8. USEMAP : Specifies the URL of a Client Side Image Map Data. See the MAP because it gives the MAP Data an Anchor Name. The Name should be included with the Name.
  9. VSPACE : Eshtablishes a Margin of White Spaces (in px.) to the Top and Bottom of the Image.
  10. WIDTH : Specifies the Horizontal Dimensions (in px.) of the Image. If it is not used the Image is Displayed in its Original Width It is used with the Height Attribute to Enlarge or Reduce the Image.

Syntax

<IMG [ALT="ALT"] [BORDER="n"] [HEIGHT="n"] [HSPACE="n"] [ISMAP] [LONGDESC="LONG DESCRIPTION"] [SRC="URL"] [USEMAP="URL"] [VSPACE="n"] [WIDTH="n"]> </IMG>

Example

<IMG SRC="IMAGES\ANI.GIF">
