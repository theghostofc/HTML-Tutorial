A

Anchor Tag is to be used in the BODY Tag. This is used to Define links or locations within the
same or to other documents. This is a Container Tag and thus has to be closed.

Attributes

  1. CHARSET : Specifies the Character Encoding of the Data designated by the Link .
  2. COORDS : Specifies the Coordinates of the Shape that Define a Clickable Area. Coordinates are measured from the Top-Left Corner of the Image. The DEFAULT and RECT take two points as X1,Y1,X2,Y2. The CIRCLE takes three points as X,Y,RADIUS. The POLY takes many points as X1,Y1,X2,Y2,X3,Y3,.....
  3. HREF : Specifies the Relative or Absolute Address (URL) of the file to be linked. To specify the link to a fragmented URL,'#' is used.
  4. HREFLANG : Specifies the Language used in the Document Linked to .
  5. NAME : Marks a location ( Fragmented URL ) within a Document to which the Browser can move when the corresponding link is activated.
  6. REL : Specifies Relationship Hyperlinks.
  7. REV : Specifies Reverse Relationship Hyperlinks.
  8. SHAPE :
    1. RECT : To create Rectangular Links.
    2. DEFAULT : This includes the Area on the Image on which no Link has been created.
    3. CIRCLE : To create Circular Links.
    4. POLY : To create Polygon Figures as Links.
  9. TARGET : Specifies the Frame name in which the contents of the linked file are to be displayed/loaded. The attributes of Target are the same used in the FRAME Tag.
  10. TYPE : It specifies the MIME Type of the Linked File or Object .

Syntax

<A [CHARSET="CHARSET"] [COORDS="X1,Y1,.."] [HREF="URL"] [HREFLANG="LANGUAGE"] [NAME="NAME] [REL="RELATION"] [REV="REVERSE RELATION"] [SHAPE="TYPE"] [TARGET="FRAMENAME"] [TYPE="MIME TYPE"]>
...Text/Image...
</A>

Example

<A HREF="WWW.YAHOO.COM" TARGET="ONE">
Link to YAHOO.COM
</A>
Effect
Link to YAHOO.COM
