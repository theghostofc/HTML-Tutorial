FRAMESET

This Tag contains Frame Definitions and Specifies Frame Spacing, Dimensions and Attributes. This is not to be used in the BODY Tag. The FRAME and NOFRAMES Tags are to be used in this Tag. This is a Container Tag and thus needs to be closed.

Attributes

  1. COLS : Specifies the Number and Dimensions of the Vertical Frames within the Current Frameset. Set COLS to a Comma Separated List of numbers to indicate the Width of each frame. Use '*' to Represent a Variable Width .
  2. ROWS : Specifies the Number and Dimensions of the Horizontal Frames within the Current Frameset. Set ROWS to a Comma Separated List of numbers to indicate the Height of each frame. Use '*' to Represent a Variable Height .

Syntax

<FRAMESET [COLS="COLS][ROWS="ROWS]>
<FRAME>
</FRAMESET>

Example

<FRAMESET ROWS="40%,60%">
<FRAMESET COLS="40%,60%">
<FRAME NAME="ONE" SCROLLING="AUTO" SRC="frameup.htm">
<FRAME NAME="TWO" SCROLLING="AUTO">
</FRAMESET>
<FRAMESET COLS="50%,50%">
<FRAME NAME="THREE" SCROLLING="YES" NORESIZE SRC="framedown.htm">
<FRAME NAME="FOUR" SCROLLING="NO" NORESIZE>
</FRAMESET>
Effect
Before Looking the Effect, Please refer to the Frame Tag.
