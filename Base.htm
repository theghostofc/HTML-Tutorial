BASE

This is to be used in the HEAD Tag. This identifies the location where all the relative URLs in a document originate. This is not a Container Tag and thus need not be closed.

Attributes

  1. HREF :
    1. Specifies the Path of the Base Document .
  2. TARGET :
    1. Identifies in which Named Frame to load the document.

    Please refer to the FRAME Tag for more information about TARGET Attribute

Syntax

<BASE [HREF="URL"] [TARGET="NAME"]>

Example

<BASE HREF="HTTP://WWW.YAHOO.COM/" TARGET="NAME">
Effect
The Effect can't be shown on the Document.
