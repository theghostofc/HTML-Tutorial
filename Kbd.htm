KBD

This is to be used in the BODY Tag. The text written in this tag is displayed as a keyboard text ie. Keystrokes by the Browser. This is a Container Tag and thus has to be closed.

Syntax

<KBD> ...Text / Keystroke... </KBD>

Example

<KBD> This is a Keystroke. CTRL+S. </KBD>
Effect
This is a Keystroke. CTRL+S.
