OBJECT

This is to be used in the BODY Tag. This Tag Embeds a Software Object into a Document.
This can be an ActiveX Control, Quick Time Movie, etc. Use the PARAM Tag in this Tag to
supply the required Parameters to the Embedded Object. If there are no parameters this Tag can be omitted. Messages and other Tags can be placed in Between the Tags for Browsers that don't support Embedded Objects. This is a Container Tag and thus has to be closed.

Attributes

  1. ALIGN : Indicates how the Embedded Object lines up relative to the edges of the browser window or other elements of the Browser Window.
    1. LEFT : Left of the Browser Window.
    2. RIGHT : Right of the Browser Window.
    3. CENTER : Center of the Browser Window.
    4. TEXTTOP : Aligns the Top of the Embedded Object with the top of the Surrounding Text.
    5. TEXTMIDDLE : Aligns the Top of the Embedded Object with the middle of the Surrounding Text.
    6. TEXTBOTTOM : Aligns the Top of the Embedded Object with the bottom of the Surrounding Text.
    7. BASELINE : Aligns the Bottom of the Embedded Object with the baseline of the Surrounding Text.
    8. MIDDLE : Aligns the Bottom of the Embedded Object with the middle of the Surrounding Text.
  2. BORDER : Indicates the Width (in px.) of a Border around an Embedded Object.
  3. CODEBASE : Specifies the Absolute or Relative URL of the Base Directory in which the Browser will look for the Data and other Implmentation Files.
  4. CODETYPE : Specifies the MIME Type of the Embedded Object's Code.
  5. CLASSID : Specifies the URL of the Embedded Object Resource.
  6. DATA : Specifies the Absolute or Relative URL of the Embedded Object's Data.
  7. DECLARE : Defines the Embedded Object without actually loading it into the Document.
  8. HEIGHT : Specifies the Vertical Dimensions (in px.) of the Embedded Object.
  9. HSPACE : Specifies the Size of Margins (in px.) to the Left and Right of the Embedded Object.
  10. NAME : Specifies the Name of the Embedded Object.
  11. SHAPES : Specifies that the Embedded Object has Shaped Hyperlinks ie. Image Maps.
  12. STANDBY : Specifies a message that the Browser Displays while loading the Object.
  13. TYPE : Specifies the MIME Type of the Embedded Object.
  14. USEMAP : Specifies the Absolute or Relative URL of a CLient Side Image Map to be used with the Embedded Object.
  15. VSPACE : Specifies the Size of Margins (in px.) to the Top and Bottom of the Embedded Object.
  16. WIDTH : Specifies the Horizontal Dimensions (in px.) of the Embedded Object.

Syntax

<OBJECT [ALIGN="ALIGNMENT"] [BORDER="n"] [CODEBASE="URL] [CODETYPE="MIMETYPPE"] [DATA="URL"] [DECLARE] [HEIGHT="n"] [HSPACE="n"] [NAME="NAME"] [SHAPES] [STANDBY="TEXT"] [TYPE="MIMETYPE"] [USEMAP="URL"] [VSPACE="n"] [WIDTH="n"]>
<PARAM>
....
</OBJECT>

Example

<OBJECT DATA="IMAGES\ANI.GIF" TYPE="IMAGE/GIF" BORDER=1 HIEGHT=100 WIDTH=100>
</OBJECT>
