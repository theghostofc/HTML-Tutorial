ADDRESS

This is to be used in the BODY Tag. The text written in this tag is displayed as an Address by the Browser. The address can be an e-mail Address or any other Street Address. This is a Container Tag and thus has to be closed.

Syntax

<ADDRESS> ...Text... </ADDRESS>

Example

<ADDRESS> ABC@XYZ.COM </ADDRESS>
Effect
This will display 'ABC@XYZ.COM' as
ABC@XYZ.COM
