BUTTON

This is to be used in the FORM Tag. This sets up a button to Submit or Reset a form as well as to activate a script. This is a Container Tag and thus need to be closed.

Attributes

  1. DISABLED : Denies Access to the Input Method.
  2. NAME : Gives a Name to the value you pass to the Form Processor.
  3. TYPE : Indicates the kind of the Button to create.
    1. SUBMIT : Creates a button which, when selected, Submits all the NAME-VALUE Pairs to the Form Processor.
    2. RESET : Sets all the Input Methods to Empty or Default Settings .
    3. BUTTON : Creates a BUTTON with No Specific Behavior .
  4. VALUE : Sets the Default Value for the Button .

Syntax

<BUTTON TYPE="TYPE" NAME="NAME" VALUE="VALUE" [DISABLED]> ...Text/Image... </BUTTON>

Example

<BUTTON TYPE="SUBMIT" NAME="BUTTON1" VALUE="SUBMIT" DISABLED> ...Text/Image... </BUTTON>

<BUTTON TYPE="SUBMIT" NAME="BUTTON1" VALUE="SUBMIT" > ...Text/Image... </BUTTON>
Effect


NOTE : To Show an Image Over a Button Instead of specifying ...Text..., specify an Image using Img Tag.
